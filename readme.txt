Automatically generate alt tags for images using TensorFlow.js and MobileNet.

== Description ==

AI Image Alt Tag Generator uses machie learning to automatically generate alt tags for images uploaded to your WordPress media library. It uses TensorFlow.js and MobileNet for image recognition, providing accurate and meaningful descriptions for your images.

Features:
* Automatic alt tag generation
* Runs a vison machine learnign model locally in the browser
* Uses TensorFlow.js + MobileNet for image recognition
* Simple accept/skip interface
* Lightweight and optimized
* Works with existing media library

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-image-alt-tag-generator` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Upload images as normal and you'll be prompted to generate alt tags

== Frequently Asked Questions ==

= How accurate are the generated alt tags? =

The alt tags are generated using MobileNet, which can recognize common objects and scenes with good accuracy. However, it's recommended to review the generated tags for critical images.
